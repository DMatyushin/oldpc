Cheat Sheet

HTML tags

Tag Name Description
table Table Обертка для всех HTML таблиц
thead Table head Определяет заголовки столбцов в таблице.
tbody Table Body Содержит данные таблицы (строки)
tr Table Row Строка таблицы
td Table Data Ячейка таблицы
tfoot Table Footer Определяет нижний колонтитул в таблице
p Paragraph Новый абзац
h Header Заголовок. Размеры от малого к большому: h6 - h1
ol Ordered List Структурированный список
ul Unsorted List Неструктурированный список

CSS attributes

Attribute Name Value
color Цвет текста Цвет в формате #FFFFFF или rgb(0,0,0)
font-family Шрифт Verdana, 'Arial Light'
font-size Размер шрифта 12px
height Высота 150px || 100em
width Ширина 150px || 100em
font-weight Насыщенность текста bold || 900
